GREEN-API Test Page

Статическая веб-страница для тестового задания: вызов методов GREEN-API и вывод ответа.

Демо

GitHub Pages: https://shidaisan.github.io/green-api/

Файлы

index.html — разметка страницы (поля ввода, кнопки, поле ответа)

style.css — стили страницы

app.js — логика вызовов API (fetch) и вывод результата

Если стили не применяются: проверь, что имя файла совпадает со ссылкой в HTML.
Сейчас должно быть либо style.css в HTML, либо файл styles.css.

Что умеет страница

Кнопки вызывают методы:

getSettings — получение настроек инстанса

getStateInstance — получение состояния инстанса

sendMessage — отправка текстового сообщения

sendFileByUrl — отправка файла по ссылке

Ответы отображаются в правом read-only поле в формате JSON (с кодом ответа HTTP).

Как использовать

Создай инстанс в GREEN-API и подключи WhatsApp номер (QR).

Открой страницу (demo или локально).

Заполни:

idInstance

ApiTokenInstance

Для отправки сообщений:

chatId (номер, например 77771234567; страница может добавить @c.us автоматически)

message (текст)

Для отправки файла:

chatId

urlFile (ссылка на файл, начинается с http(s)://)
